# Telegram Chat Export Participants Bot

Бот принимает файлы экспорта истории чата Telegram (JSON/HTML), извлекает уникальных авторов сообщений и отдельно собирает упоминания (@username).
Данные не сохраняются: обработка "на лету", результат отправляется пользователю.

## Возможности
- Приём до 10 файлов экспорта
- Извлечение участников (по авторам сообщений)
- Упоминания (@username) отдельно
- Если участников < 50 — отправка списком
- Если участников >= 50 — отправка Excel (Participants + Mentions)

## Ограничения
- В стандартном экспорте Telegram обычно нет Bio/About — колонка остаётся пустой.
- Username автора может отсутствовать в экспорте (особенно в HTML), тогда выводится имя.
